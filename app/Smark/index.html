Installation

git clone git@github.com:24377965mjGithub/smark.git

or download the zip file here

For Laravel

Paste the Smark folder inside the app/ folder.

Create a service provider

Go to the terminal and type: php artisan make:provider SmarkServiceProvider

Register the Smark class on the SmarkServiceProvider. 

$this->app->singleton(SmarkServiceProvider::class, function ($app) {
        return Smark::class;
});

Basic Usage

Computation

Syntax:
Smark::compute(method, array of numbers (as many as you want))

Addition

Smark::compute('add', [2, 3, 4, 5])

Output: 14

Subtraction

Smark::compute('minus', [5, 4])

Output: 1

Multiplication

Smark::compute('multiply', [5, 4])

Output: 25

Division

Smark::compute('divide', [4, 2])

Output: 2

isEven()

Syntax:
Smark::isEven(number)

Smark::isEven(24)

Output: true or 1

linearRegression()

Syntax:
Smark::linearRegression(xAxis array, yAxis array, result type)

Smark::linearRegression([1, 4, 3, 4, 5, 6], [2, 3, 4, 5, 6, 8], 'slope')

Output: 1.1235955056179776

Smark::linearRegression([1, 4, 3, 4, 5, 6], [2, 3, 4, 5, 6, 8], 'intercept')

Output: 0.3595505617977525

Smark::linearRegression([1, 4, 3, 4, 5, 6], [2, 3, 4, 5, 6, 8], 'all')

Output: {"slope":1.1235955056179776,"intercept":0.3595505617977525}

JSON File CRUD

A JSON file CRUD (Create, Read, Update, Delete) operation involves managing data stored in JSON file. This set of functions allows you to perform basic data operations, such as creating new entries, reading existing data, updating specific records, and deleting unwanted entries.

In this case, I created a JSON file inside the public/json folder. public/json/sample.json

jsonRead()

Syntax:
Smark::jsonRead(json file directory) 

Smark::jsonRead('json/sample.json')

Output: []

jsonPush()

Syntax:
Smark::jsonPush(json file directory, data)

Adding a new value to the end of the array. 
Smark::jsonPush('json/sample.json', ['id' => 1, 'name' => 'Mark Jason'])

Output: [{"id": 1, "name": "Mark Jason"}]

jsonUnshift()

Syntax:
Smark::jsonUnshift(json file directory, data)

Adding a new value in the beginning of the array. 
Smark::jsonUnshift('json/sample.json', ['id' => 0, 'name' => 'Mark Jason'])

Output: [{"id": 0, "name": "Mark Jason"}, {"id": 1, "name": "Mark Jason"}]

jsonDelete()

Syntax:
Smark::jsonDelete(json file directory, key, target value (string, integer or float))

Deleting an item in the array. 
Smark::jsonDelete('json/sample.json', 'id', 0)

Output: [{"id": 1, "name": "Mark Jason"}]

jsonUpdate()

Syntax:
Smark::jsonUpdate(json file directory, key, value (string, integer or float), target key, target value (string, integer or float))

Update an item in the array. 
Smark::jsonUpdate('json/sample.json', 'id', 1, 'name', 'Mark Jason Updated')

Output: [{"id": 1, "name": "Mark Jason Updated"}]

Other available functions

calculateTotalPrice()

A function that calculates the total price of an order after applying discounts and tax.

Syntax:
Smark::calculateTotalPrice([array of orders ('price', 'quantity')], discount threshold, discount (percentage to decimal), tax (percentage to decimal)) 

Smark::calculateTotalPrice([
    ['price' => 20, 'quantity' => 3],  // $60
    ['price' => 50, 'quantity' => 2]   // $100
], 100, 0.10, 0.05)

Output: 151.2

Explanation: If the subtotal exceeds $100, apply a 10% discount. Apply a 5% tax on the discounted subtotal.

calculateBMI()

A function that calculates the Body Mass Index (BMI). The function takes two arguments: weight (in kilograms) and height (in meters).

Syntax:
Smark::calculateBMI(weight (kilograms), height (meters)) 

Smark::calculateBMI(70, 1.5)

Output: 31.11

Explanation: The BMI is calculated using the formula: BMI = weight / (height * height). The result is rounded to two decimal places for better readability.
